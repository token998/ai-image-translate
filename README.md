# AI图片翻译不限次数可二次修改
* 本地模型运行，不限翻译次数免费使用，支持批量处理
* 可二次编辑修改，内置多种字体和特效，可修改颜色、字体、字号、样式等丰富的内容
* 背景可二次扣图，可涂抹擦除不要的内容，可使用画笔
* 支持繁体中文、英语、日语、韩语、越南语、泰语、印尼语、俄语、阿拉伯语、西班牙语、意大利语、法语等几十种主流语言
# 实现原理
基于百度paddleocr大模型及lama图片模型，本地运行无需服务器成本，因此不限次数使用
